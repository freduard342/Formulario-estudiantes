Formulario de ingreso de notas

Descripción del proyecto

Esta aplicación fue desarrollada para registrar y calcular las calificaciones de un estudiante de manera automática. El sistema permite ingresar cuatro notas iniciales, calcular el promedio y determinar si el estudiante aprueba o reprueba.

En caso de que el promedio no sea suficiente:

El programa habilita un campo adicional para ingresar la nota del completivo.

Si luego del completivo el estudiante aún no alcanza el promedio requerido, se habilita otro campo para ingresar la nota del extraordinario.

Al finalizar, la aplicación muestra el promedio final y el estado académico del estudiante, todo de forma clara y automática.

Herramientas utilizadas

Lenguaje de programación: C#

Plataforma: .NET Framework

Entorno de desarrollo: Visual Studio 2022
